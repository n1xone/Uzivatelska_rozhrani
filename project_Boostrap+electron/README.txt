UniBudget Pro Desktop/Web - URO Bootstrap projekt
Autor: Nikita Chervinskyi (CHE0086)
Cvičící: ZUR0037

Co projekt obsahuje:
- Bootstrap 5 web aplikaci v index.html
- Electron desktop wrapper: package.json + electron-main.js
- CRUD transakcí: přidání, editace, smazání
- CRUD studentů podle Qt projektu
- filtrování, vyhledávání, CSV export
- grafy přes Chart.js
- dark/light theme
- animace, modální okna, toast notifikace
- localStorage ukládání dat
- menu / O autorovi

Spuštění jako web:
1) Otevřete index.html v prohlížeči.

Spuštění jako desktop aplikace Electron:
1) Nainstalujte Node.js
2) Ve složce projektu spusťte:
   npm install
   npm start

Poznámka:
Projekt nepoužívá build složku. Všechny projektové soubory jsou relativně ve složce projektu.
